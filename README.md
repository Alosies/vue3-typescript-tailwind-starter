# Vue 3, Vite, TypeScript & Tailwind CSS Starter

> A simple, Vue 3 starter with TypeScript support

## Features

- Vue 3
- TypeScript
- Tailwind CSS
- Very simple store
- Eslint

## Project setup and usage

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```
